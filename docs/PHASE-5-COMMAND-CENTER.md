## 🎓 KNOWLEDGE BASE

### Understanding P2P in MyKit

**The Problem We're Solving**:
- Field workers offline, need to share scans
- Cloud sync is delayed (wait for connectivity)
- P2P lets them sync instantly when nearby

**The Challenge**:
- Two devices both edited same note offline
- P2P wants to sync, but which version wins?
- Need OT algorithm (smart merge, no data loss)

**The Solution**:
- Operational Transform: Merge edits automatically
- QR pairing: Only pair with devices you trust
- TLS encryption: Secure in-transit
- Network lab: Test all failure modes

### Why This Matters

**Enterprise Value**:
- ✅ Offline collaboration (no cloud needed)
- ✅ 10x faster transfers
- ✅ Privacy (data stays local)
- ✅ Compliance-friendly (no cloud transmission)

**Risk**:
- ❌ OT complexity (can cause data loss if wrong)
- ❌ Auth bugs (peer spoofing if weak)
- ❌ Web gap (no native P2P in browsers)
